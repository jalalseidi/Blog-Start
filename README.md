# Flask Blog App

A simple Flask blog application that fetches blog posts from an external API and displays them using dynamic routes.

## Features
- Fetches blog data from npoint API
- Converts JSON data into Python objects
- Displays all blog posts on the home page
- Shows individual blog posts using dynamic URLs

## Technologies Used
- Python
- Flask
- Requests
- Jinja2
